# COMART API

> Backend API for comart, an ecommerce platform

## Getting Started

### 1. Clone the repository

```
git clone https://github.com/mikiasworash/comart-api.git

```

### 2. Install Dependencies

```
cd comart-api
npm install
```

### 3. Configure .env file (update the environment variables)

### 4. Start the API

```
npm run dev
```
